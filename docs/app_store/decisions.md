# App Store Decisions

## Release model

- Distribution: Paid upfront
- Price: `$5.99`
- Category: `Utilities`
- Secondary category: none
- In-App Purchases: none
- Subscriptions: none

## Privacy position

- No account
- No ads
- No tracking
- No third-party analytics
- No data collection sent off-device
- User display settings are stored locally on-device

## Localization strategy

- Default App Store language: `en-US`
- Initial localized language: `ja-JP`
- Root public support pages stay English
- Japanese public pages live under `/ja/`

## Availability strategy

- Target: as many countries and regions as possible
- Default release posture: enable all countries/regions unless App Store Connect requires region-specific compliance input

## Review assumptions

- Expected age rating: `4+`
- Core message: reuse old iPhone / iPad devices as an offline clock and calendar

## Naming note

- Requested App Name: `Old iOS Clock & Calendar`
- Fallback if App Review flags trademark wording: `Old Device Clock & Calendar`
