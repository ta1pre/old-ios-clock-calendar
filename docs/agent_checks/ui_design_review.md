# UI Design Review

置き時計UIプロトタイプのレビュー観点です。

## Goal

このUIは「古いiPhone / iPad / iPad miniを横向きに置いて、左の時計表示と右のカレンダーを見る」ためのものです。

## Must

- 左に時計表示がある
- 右に月間カレンダーがある
- 今日の日付だけが青い強調で分かる
- 横置き固定の前提になっている
- メイン画面で設定UIが主役になっていない
- iPhone横向きでもカレンダーが読める
- iPad / iPad miniでも左右レイアウトが間延びしない
- 設定画面が `Light` / `Dark` と週始まりだけに絞られている

## Should

- アナログ時計は正円を維持する
- アナログ時計の数字と目盛りが読みやすい
- 赤い秒針が細すぎない
- カレンダーは7列 x 最大6行で安定させる
- ダーク表示で、夜間に眩しすぎない
- 背景と通常日のコントラストは控えめにする
- Auto Layoutで端末サイズ差に対応しやすい構成にする
- 設定画面は横向きのまま読みやすい

## Output Format

レビュー結果は以下の順でまとめる。

1. UI方針の評価
2. Must違反の有無
3. 改善した方がよい点
4. 次に進んでよいか
5. 保留すべき確認事項
